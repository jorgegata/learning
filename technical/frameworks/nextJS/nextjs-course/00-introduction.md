# NextJS course

This course is achieved by building a complete full-stack web application.

It is a financial dashboard that contains:

* Public home page
* Login page
* Dashboard pages that are protected by authentication
* The ability for users to add, edit, and delete invoices

It has an accompanying database, which you will set up in a later chapter.

## Overview

After this project, you will have the necessary core skills to implement full-stack applications. This contains:

* **Styling:** different ways to style
* **Optimization:** images, links, and fonts
* **Routing:** create nested layouts and pages using file-system
* **Data Fetching:** set up a postgres database, and best practices for fetching and streaming
* **Search and Pagination:** search and pagination using URL search params
* **Mutating data:** how to mutate data using React Server Actions, and revalidate the Next.js cache
* **Error handling:**
* **Form validation and Accessibility:** server-side form validation and tips for improving accessibility
* **Authentication:** how to add authentication to the application
* **Metadata:** how to add metadata and prepare application for social sharing
